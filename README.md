# Setup

```bash
$ git clone git@github.com:frosas/dotfiles.git
$ cd dotfiles
$ git submodule update --init
$ ./setup
```

## Manual steps

In ~/.bashrc

```bash
source ~/.../dotfiles/bash/.bashrc
export CDPATH=.:~/projects
```
